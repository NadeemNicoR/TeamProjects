<h2 id="page-title">Advanced Topics in Machine Learning</h2>
<p>This web page provides information on the course <em>Advanced Topics in Machine Learning</em> (summer term 2018). The course deals with selected topics of Machine Learning, including:</p>
<ul>
<li>Support Vector Machines</li>
<li>Semi-Supervised Learning (semi-supervised classification and clustering)</li>
<li>Dealing with massive datasets</li>
</ul>
<p>Prerequisite for attending this course is a basic knowledge of computer science, especially in Machine Learning. Programming skills are an advantage concerning the practical exercises.</p>

<h4>Exercise Classes <a id="anker_nolinkzeichen" title="exercises" name="exercises"></a></h4>
<p>The exercise classes have two objectives. First, regular assignments concerning the theory taught in the lecture will be given (about one week in advance). These have to be prepared by the students and are then discussed during class. Secondly, the lecture will be accompanied by a software project. Its goal is to practice the implementation of machine learning techniques into a larger system. This will be done as a joint group work. The development will partly be done during the exercise classes. However, further development outside the class might be necessary to complete the project. We expect active involvement of all students, both in the project and the theoretical assignments.</p>
<p><a class="kein_linkzeichen" href="https://lsf.ovgu.de/qislsf/rds?state=verpublish&status=init&vmfile=no&publishid=127662&moduleCall=webInfo&publishConfFile=webInfo&publishSubDir=veranstaltung">Please register yourself via LSF for the exercise classes.</a></p>
<h4>Requirements for Class Fulfillment <a id="anker_nolinkzeichen" title="fulfillment" name="fulfillment"></a></h4>
<p>At the end of the course, there will be an oral exam. As a prerequisite, we expect active involvement both during the exercise and in the software project.</p>

<h4>Frequently Asked Questions&nbsp;<a name="faq"></a></h4>
<ul>
<li><strong>Question:&nbsp;</strong>Do I have to register for the course? If so, where?<br /><strong>Answer: </strong>Registrations are done via LSF (link above). The registrations will be open from the 19th of March 2018 until the 23rd of April.</li>
<li><strong>Question:</strong>&nbsp;I can't be there for the first exercise! What should I do?<br /><strong>Answer: </strong>Please write to me (Marcus Thiel) an e-mail about your absence.</li>
<li><strong>Question:&nbsp;</strong>Do I really have to do the first assignment for the FIRST exercise?<br /><strong>Answer:&nbsp;</strong>Yes, of course. The questions are easy enough to be handled without having heard the lecture.</li>
<li><strong>Question:&nbsp;</strong>One exercise seems hardly enough. Shouldn't there be more?<br /><strong>Answer:&nbsp;</strong>Additional exercises will be planned based on the number of interested students. Most likely in or after the first lecture.</li>
<li><strong>Question:&nbsp;</strong>What's that thing about "voting"?<br /><strong>Answer:&nbsp;</strong>It means, that you have to prepare the assignments BEFORE the exercise, in which they are discussed in. A sheet will be given around, where you have to mark all assignments, that you were able to solve. Marking an assignment means, that you are able to present your solution in front of the class. I expect everyone to at least present once, but twice would be best. Your solution does not have to be correct, but if you are not able to explain the assignment at all, all marked tasks for the particular exercise will be removed from your votings.</li>
<li><strong>Question:&nbsp;</strong>How many "votings" do I need in order to get my exam admission?<br /><strong>Answer:&nbsp;</strong>The absolute minimum is half of all the assignments. Every task on each assignment sheet counts as a single point. So you have to solve half of these tasks. BUT I expect active involvement in more than half of the tasks. Please try to solve at least 2/3.</li>
<li><strong>Question:&nbsp;</strong>Is this all I need for my admission?<br /><strong>Answer:&nbsp;</strong>No! There is still the project, which has to be completed until the end of the exercise classes. (a bit earlier than that) Exact dates will be given in the first exercises.</li>
<li><strong>Question:&nbsp;</strong>What is that project?<br /><strong>Answer: </strong>The actual project will be given in the first or second exercise. It will be about data analysis and will involve a certain amount of programming, The latter can be greatly reduced by using existing libraries and tools.</li>
<li><strong>Question:</strong>&nbsp;But I'm not from a Computer Science background / I can't program! Can I do something else instead?<br /><strong>Answer:&nbsp;</strong>No, the project will be equal for everyone. The way you solve it can differ though.</li>
<li><strong>Question:&nbsp;</strong>I have never heard anything about Data Mining and/or Machine Learning before. Should I still visit this course?<br /><strong>Answer:&nbsp;</strong>Only if you are willing to fill the gaps yourself. This course requires a certain basic understanding about Machine Learning techniques. You don't need to be an expert though. You can have a look at the literature at the <a title="Machine Learning" href="/findke/en/Studies/Courses/Past+Terms/Winter+Term+2017_2018/Machine+Learning.html">Machine Learning course</a>.</li>
<li><strong>Question:</strong>&nbsp;My math skills are very rusty, do I need them for this course?<br /><strong>Answer:&nbsp;</strong>Same as the Machine Learning question: You need to have basic skills (e.g. derivations, matrices, vectors) in order to understand this course. Those things will only partially be explained again, so you should be ready to fill the gaps yourself.</li>
<li><strong>Question:&nbsp;</strong>I really, really like Machine Learning, but the assignments are way too difficult and time consuming! Could we please reduce the amount of work?<br /><strong>Answer:&nbsp;</strong>At the moment: No. The assignments may change depending on the feedback though.</li>
<li><strong>Question:&nbsp;</strong>How will the exam look like? Will it be oral or written?<br /><strong>Answer:&nbsp;</strong>Currently, the exam is oral. This can be changed though depending of the number of students, that will take the course. Depending on that, the exam will look different. Since there has not been a written exam, yet, an example can't be given at this point of time.</li>
</ul>
