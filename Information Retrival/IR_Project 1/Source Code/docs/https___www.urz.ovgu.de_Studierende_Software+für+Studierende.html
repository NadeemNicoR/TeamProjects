<h2 id="page-title">Software für Studierende</h2>
<p>Empfehlungen f&uuml;r Studierende (und Mitarbeiter):</p><p>Die durch das URZ abgeschlossenen Vertr&auml;ge zum Erwerb von Software gelten nicht f&uuml;r Studierende. Ausnahmen bilden die Programme Microsoft Dreamspark, Labview, Scientific Word, Sophos Anti-Virus und Citavi.<br> Sophos Anti-Virus, MATLAB und Citavi kann auch auf den Heim-Rechnern der Mitarbeiter/-innen und Studierenden<br> installiert werden. Die Programme Microsoft Dreamspark, Labview und Scientific Word k&ouml;nnen die Studierenden von der entsprechenden Lehrkraft, f&uuml;r die die Programme beschafft wurden, erhalten.</p><p>Empfehlenswert ist nicht nur nach dem Produkt sondern auch nach der Herstellerfirma einer Software zu suchen. Eine gute Quelle f&uuml;r Software sind unter Umst&auml;nden auch gro&szlig;e Interneth&auml;ndler.</p><h3>Autodesk</h3><p>Das&nbsp;<a href="https://www.autodesk.com/education/home" target="_blank">Portal von Autodesk</a> f&uuml;r <strong>Design- und Engineering</strong> Studenten bietet neben kostenlosen Downloads der neuesten Autodesk Software auch Lernprogramme, eine Konstruktions- Bibliothek, Tipps &amp; Tricks uvm.</p><h3>Citavi</h3><p><a title="Citavi" href="/Unsere+Leistungen/Anwendungen/Campuslizenzen/Citavi.html">Citavi</a> verwaltet Literaturquellen, organisiert Zitate, unterst&uuml;tzt die Internetrecherche f&uuml;r Literatur, hilft beim Wissen- und Aufgabenmanagement u. a.</p><h3>Journey Education Europe</h3><p>&Uuml;ber <a class="undefined" title="Journy Education Europe" href="http://www.journeyed.com/">Journey Education Europe</a> k&ouml;nnen Studenten &amp; Dozenten - Software zu Studentenpreisen beziehen.</p><h3>EndNote</h3><p>Studentenlizenzen f&uuml;r Endnote k&ouml;nnen unter anderem im Webstore von <a href="http://directory.adeptscience.de/productid/I-EN-ED-STUD/2/474/I-EN-ED-STUD.html"><strong>alfasoft </strong></a>kosteng&uuml;nstig bestellt werden.</p><h3>Flexpro</h3><p>F&uuml;r Studierende stellt die Weisang GmbH eine s.g. <a href="http://www.weisang.com/produkte/flexpro-10/studenten-und-hochschulen/studentenformular/"><span class="externalLink"><span class="linkextern">FlexPro 10 &Uuml;bungslizenz</span></span></a><strong>gratis</strong> zur Verf&uuml;gung.&nbsp;&nbsp;&nbsp;&nbsp;</p><h3>FreeCode</h3><p><a title="FreeCode" href="http://freecode.com/">FreeCode</a>&nbsp;bietet jede Menge <strong>Open Source</strong> Software zum Herunterladen.</p><h3>GNU Image Manipulations Programm</h3><p>Das <a title="The GIMP" href="http://www.gimp.org/" target="_self">GNU Image Manipulation Program</a> (<strong>GIMP</strong>) ist ein professionelles <strong>Bildbearbeitungsprogramm</strong>. Diese Freeware steht den bekannten Bildbearbeitungsprogrammen in nichts nach. Auch gibt es eine <a class="undefined" title="Gimp-Doku" href="http://docs.gimp.org/de/" target="_self">deutschsprachige Dokumentation</a>.</p><h3>KNOPPIX</h3><p><a title="KNOPPIX" href="http://www.knopper.net/knoppix/">KNOPPIX</a>&nbsp;beinhaltet eine komplette <strong>Linux Distribution</strong> auf CD/DVD zum Herunterladen und/oder Bestellen.&nbsp; Office, Surfen, Spielen und Arbeiten sind ohne Installation m&ouml;glich.</p><h3>MATLAB</h3><p>Studierende und Mitabeiter/-innen k&ouml;nnen MATLAB, Simulink und alle begleitenden Toolboxen f&uuml;r den Einsatz auf eigener und universit&auml;rer Hardware sowie begleitendes, umfangreiches Training kostenfrei &uuml;ber unser Portal bei <a href="https://www.mathworks.com/academia/tah-portal/otto-von-guericke-universitat-magdeburg-40693800.html">MathWorks</a> erhalten.</p><h3>Maxima</h3><p><a class="undefined" title="Maxima-Quelle" href="http://maxima.sourceforge.net/" target="_self">Maxima</a> ist ein kostenloses Computer Algebra System (wie Derive, Maple, Mathematica u.a.). Es bietet diverse Funktionalit&auml;t (Numerik, Vereinfachen von Termen, Arbeit mit Gleichungen (L&ouml;sen, Vereinfachen, Umstellen), Arbeit mit Funktionen, 2D und 3D Grafiken, Stochastik/Statistik, Differential- und Integralrechnung, Vektoren- und Matrizenrechnung usw.) Ein deutscher <a class="undefined" title="Maxima-Kurs" href="http://www.austromath.at/daten/maxima/index.htm" target="_self">Maxima-Online-Kurs</a> wird ebenfalls angeboten.</p><h3>Microsoft Office</h3><p><a title="Office 365 University" href="https://www.microsoft.com/de-de/store/d/office-365-university/cfq7ttc0k5bb?activetab=pivot%3aoverviewtab">Microsoft Office 365</a> wird von Microsoft als <a class="undefined" href="https://www.microsoft.com/de-de/p/office-365-university/cfq7ttc0k5bb">Sonderangebot f&uuml;r Studenten</a>&nbsp;angeboten. Das kostenlose Office 365 f&uuml;r Studenten kann von der OVGU nicht angeboten werden, da es nicht Inhalt unseres Microsoft-Lizenzvertrages ist.</p><h3>Microsoft Studentoptions</h3><p>Microsoft Software kann &uuml;ber unseren Microsoft-Bezugspartner <a title="cancom" href="http://shop.cancom.de/online/wa?showtemplatecomponent=startseite_cancom&amp;menulevel1=shop&amp;shop=cancom" target="_self">Cancom</a>&nbsp;bezogen werden. Alternativ ist es f&uuml;r Studenten auch m&ouml;glich, Microsoft Software &uuml;ber <a href="https://www.studyhouse.biz/cgi-bin/home">studyhouse</a> g&uuml;nstig zu erwerben.</p><h3>Dreamspark / Microsoft Imagine</h3><p>Vorsicht, dieses Anmeldeformular ist nur aus dem Universit&auml;tsnetz zu erreichen.</p><p><a title="Microsoft Imagine (Dreamspark)" href="/Unsere+Leistungen/Anwendungen/Campuslizenzen/Microsoft+Imagine+%28Dreamspark%29.html">zur Anmeldung<strong><br></strong></a>kostenfreie Software von Microsoft, Dremspark Standard oder Dreamspark Premium entsprechend der Studienrichtung</p><h3>Octave</h3><p><a href="http://www.gnu.org/software/octave/">Octave</a>&nbsp;ist eine freie zu empfehlende Software, die weitgehend kompatibel zu MATLAB ist. Wer nicht auf alle Funktionalit&auml;ten von MATLAB verzichten m&ouml;chte, kann jetzt eine Vollversion &uuml;ber unser <a title="MATLAB" href="https://de.mathworks.com/academia/tah-portal/otto-von-guericke-universitat-magdeburg-40693800.html">Portal</a> bei Mathworks erhalten.</p><h3>LibreOffice</h3><p><a href="https://de.libreoffice.org/" target="_blank">LibreOffice</a> ist eine freie B&uuml;rosoftware, die u. a. Programme zur Textverarbeitung, Tabellenkalkulation, Pr&auml;sentation und zum Zeichnen beinhaltet.</p><h3>OriginPro Home Use</h3><p><a class="undefined" title="OriginPro Student Use" href="http://www.originlab.de/index.aspx?go=Products/OriginStudentVersion" target="_self">OriginPro Home Use</a> f&uuml;r Studenten ist eine Datenanalyse und Grafiksoftware (Impulsanpassung, Oberfl&auml;chenanpassung Statistik, Signalverarbeitung, Bildverarbeitung). Diese Software kann &uuml;ber den<br> DV-Beauftragten des Institus f&uuml;r 1 Jahr kostenlos bezogen werden. Falls Ihnen der DV-Beauftragte nicht bekannt ist, teilen Sie dem IT-Service-Point (<a href="mailto:it-service@ovgu.de">it-service@ovgu.de</a>) per E-Mail den Namen des Hochschullehrers oder seines Mitarbeiters - f&uuml;r dessen Fach/Lehrveranstaltung Sie OriginPro einsetzen wollen - mit. Als Antwort erhalten Sie die E-Mail-Adresse des zust&auml;ndigen DV-Beauftragten, der die Bestellung der Software in die Wege leiten wird.</p><h3>R-Projekt</h3><p><a title="r-project" href="http://www.r-project.de/">R-Projekt</a> kann als Open Source Software (kommandozeilenorientiert) f&uuml;r Statistik (Windows, Linux, Mac) empfohlen werden.</p><h3>PSPP</h3><p><a href="http://www.gnu.org/software/pspp/">PSPP</a> kann als freier Ersatz zur kommerziellen Statistiksoftware SPSS angesehen werden.</p><h3>SolidWorks Student Design Kit</h3><p><span class="egotecExceptionLink undefined">SolidWorks Student Design Kit</span> (laufzeitbegrenzte Version von SolidWorks f&uuml;r die private Nutzung) steht allen Studenten und Lehrkr&auml;ften der OVGU kostenfrei zur Verf&uuml;gung. Informationen zur&nbsp; Installation des SolidWorks Student Design Kit finden Sie unter&nbsp;&nbsp;<a href="https://elearning.ovgu.de/course/view.php?id=896">https://elearning.ovgu.de/course/view.php?id=896</a><br>(Elearning-Portal, Anmeldung mit dem OVGU-Account erforderlich)</p><h3>Software Solutions (LabView)</h3><p>ist eine Software zur Messdatenerfassung und -verarbeitung. Haupt-Anwendungsgebiete von LabVIEW sind die Mess- und Automatisierungstechnik. Bei Bedarf kann eine befristete Lizenz beim DV-Beauftragten des zugeh&ouml;rigen Institutes beantragt werden.</p><h3>SPSSStudentenversion</h3><p><a class="undefined" title="SPSS f&uuml;r Studenten" href="http://www.spss.com/de/vertical_markets/academia.htm" target="_self">SPSS Studentenversion</a> kann f&uuml;r 100 EUR und weniger von <a class="undefined" title="OntheHub" href="http://www.onthehub.com/spss/">OntheHub</a> bezogen werden.</p><h3>VMware</h3><p><span style="font-size: 9pt;">Kostenlose Software von VMware, diese kann &uuml;ber den </span><a style="font-size: 9pt;" href="https://e5.onthehub.com/WebStore/Welcome.aspx?ws=b466eb34-1ca2-de11-886d-0030487d8897">Webstore</a><span style="font-size: 9pt;"> bezogen werden.</span></p><p>&nbsp;</p><p><a href="http://store.apple.com/de-edu/browse/campaigns/education_pricing">http://store.apple.com/de-edu/browse/campaigns/education_pricing</a></p>
